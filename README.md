```

│   ├── stl/              # 标准模板库

├── java/                  # Java算法实现
│   ├── basics/           # Java基础语法
│   ├── advanced/         # Java高级特性
```

## 学习目标

- 掌握C++和Java两种编程语言

## 使用说明

### C++环境配置
- 编译器：GCC 或 Clang
- 标准：C++11 或更高版本
- 编译命令：`g++ -std=c++11 filename.cpp -o output`

### Java环境配置
- JDK版本：8 或更高版本
- 编译命令：`javac ClassName.java`
- 运行命令：`java ClassName`
